# 🖼️ image-to-speech-webapp

A simple browser-based tool to extract and narrate text from images using built-in OCR (Tesseract.js) and speech synthesis. Just paste or drop an image and click play.

## ✨ Features

* 📷 Paste, drop, or upload images with text
* 🔎 Extracts readable text using Tesseract.js OCR
* 🎤 Narrates extracted text using system speech synthesis
* 🎚 Adjustable **rate** and **pitch**
* 🎧 Select from available voices (male/female, different languages)
* 🌗 **Dark/light mode** with auto-detect and toggle
* 🧹 Clear/reset the extracted content
* ⚙️ Works entirely client-side — no backend required

## 🚀 Usage

1. Clone or fork this repo:

   ```bash
   git clone https://github.com/jajera/image-to-speech-webapp
   cd image-to-speech-webapp
   ```

2. Open `index.html` in a modern browser (Chrome, Edge, Safari).

3. Or [use the GitHub Pages version](https://jajera.github.io/image-to-speech-webapp).

## 🧠 Tech Stack

* Vanilla HTML, CSS, JavaScript
* [Tesseract.js](https://github.com/naptha/tesseract.js) for OCR
* Web Speech API (`SpeechSynthesisUtterance`)
